## 🚀 Overview
The **Email Spam Classifier** is a Machine Learning-based project that automatically classifies emails as **Spam** or **Ham (Not Spam)** using Natural Language Processing (NLP) techniques.

This system helps in filtering unwanted emails, improving productivity, and enhancing email security.

---

## 🎯 Objective
- Build an automated spam detection system  
- Apply NLP techniques for text processing  
- Train and evaluate machine learning models  
- Achieve high accuracy with minimal false positives  

---

## 🧠 Algorithms Used
- Naive Bayes  
- Logistic Regression  
- Support Vector Machine (SVM)
